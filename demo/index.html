What is Kohaku?

Kohaku is a neural episodic memory engine built on hyperdimensional computing (HDC). Concepts are encoded as 10 000-dimensional bipolar (±1) vectors. Operations like bundle (superposition), bind (composition), and cosine similarity let an LLM remember, retrieve, consolidate, and forget — across sessions, beyond context windows, with no vector database.

⚡

HDC Encoding

Deterministic LCG-seeded ±1 hypervectors. Same text → same vector. Near-orthogonal in 10k-D.

🌀

Bundle & Query

Combine vectors via majority vote; query the bundle to recall any of its members by cosine similarity.

💾

Persistence

JSON for inspection, packed-bit .hkb binary for production. ~27× smaller, bit-identical recall.

🌱

Consolidation

Greedy bundle-of-bundles clustering. Many noisy episodes → few semantic centroids.

🕰️

Ebbinghaus Decay

Exponential forgetting curves with optional floor. Recent memories dominate; old ones fade.

🦀

Rust + Python

Rust core for performance; pure-Python fallback by default. Same LCG math both sides.

Storage Format

Storing 1 000 entries (each a key + value bipolar vector at 10 000 dims). Both formats round-trip with bit-identical recall. The binary format packs each ±1 component into a single bit — no precision loss.

Binary .hkb layout

Little-endian. Each ±1 component → 1 bit (numpy.packbits).

// header (32 bytes)
magic        : b"KHKU"      // 4
version      : u16           // 2
dims         : u32           // 4
capacity     : u32           // 4
next_id      : u64           // 8
timestamp    : u64           // 8
num_entries  : u32           // 4

// per entry
id           : u64
timestamp    : u64
label_len    : u16
label_utf8   : label_len bytes
key_bits     : dims/8 bytes

The whole library, in eight lines

Encode. Bundle. Persist. Consolidate. Decay. Done.

from kohaku import (
    EpisodicMemory, encode_text,
    save, load, consolidate_to_memory,
    DecayConfig, query_with_decay,
)

mem = EpisodicMemory(capacity=1000)
key = encode_text("konjo means beautiful")
mem.store(key, key, label="amharic")

save(mem, "memories.hkb")              # packed binary, ~27× smaller than JSON
restored = load("memories.hkb")         # bit-identical recall

semantic = consolidate_to_memory(mem, similarity_threshold=0.3)
results = query_with_decay(mem, key, top_k=5,
                            config=DecayConfig(half_life=100.0))
