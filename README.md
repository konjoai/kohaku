# 🦀 Kohaku

![Language](https://img.shields.io/badge/language-rust-blue) ![License](https://img.shields.io/badge/license-busl--1.1-green) ![Status](https://img.shields.io/badge/status-active-brightgreen)

> 🚀 A rust project for Neural episodic memory engine. Stores conversation history, documents, or events as compressed HDC hypervectors. Retrieves by associative similarity, not keyword or vector search. Works as a drop-in persistent memory layer for any LLM..

## ✨ Features

- 🛠️ Modern rust development

- 📚 Well-documented and easy to use
- ✅ Production-ready code
- ⚡ Performant and reliable

## 🚀 Quick Start

### Installation

